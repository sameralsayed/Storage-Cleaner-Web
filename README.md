# Storage Cleaner Web

🧹 A beautiful and functional web version of a phone storage cleaner. Analyze storage, remove duplicate photos, compress videos, and free up space instantly.

![Storage Cleaner](https://via.placeholder.com/800x400/28a745/ffffff?text=Storage+Cleaner)

## ✨ Features
- 📊 **Storage Analyzer** — realistic breakdown and usage visualization
- 📸 **Duplicate & Similar Photos** — click to select and remove duplicates
- 🎥 **Video Compressor** — simulate high-quality video compression
- ⚡ **Deep Clean** — one-tap smart optimization
- 📜 **Clean History** — logs every action (saved in browser)
- 📱 **Fully Responsive** — excellent experience on mobile, tablet, and desktop
- 🔒 **100% Private** — everything runs locally in your browser

## 🚀 How to Use
1. Open `index.html` in any modern browser
2. Check the storage analyzer
3. Select duplicate photos by tapping/clicking them
4. Use Deep Clean or individual tools
5. All actions are safe simulations

## 🛠️ Built With
- HTML5
- CSS3
- Bootstrap 5
- jQuery
